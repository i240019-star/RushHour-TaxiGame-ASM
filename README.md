## **Features**
- **Dynamic 20√ó20 Board** with roads, buildings, passengers, NPC cars, and obstacles  
- **Multiple Game Modes**: Career Mode, Time Mode, Endless Mode  
- **Taxi Selection**: Red Taxi or Yellow Taxi (or random selection)

- **Leaderboard**: Stores top 10 player scores in `highscores.txt`  
- **Pause & Continue**: Save game state and resume later  
- **Audio Feedback**: Sounds for clicks, pickups, drops, crashes, and game end  
- **NPC Car Movement**: Other cars move automatically, speed increases as you progress  
- **Difficulty Levels**: Easy, Normal, Hard
